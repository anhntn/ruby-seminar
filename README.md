プログラミング初心者向けRuby勉強会「Ruby入門以前」
=
2014年04月07日にMozilla Japan ファクトリスペースにて行う、プログラミング初心者のためのRuby勉強会です。この勉強会の目的は、Rubyの基本を理解し、ウェブサイトや本を使って自分でRubyを学習していけるようにすることです。
